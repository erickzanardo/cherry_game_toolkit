Reusable game progress based on sequencial levels logic

## How to use

Add the cubit to your project

```yaml
BlocProvider(
  create: (context) => GameProgressCubit(maxLevel: 10),
  child: YourGameWidget(),
),
```

Use the cubit methods and state to manage the game progress

```dart
final cubit = context.read<GameProgressCubit>();

cubit.state // Current level
cubit.advanceLevel(); // Advance to the next level
cubit.reset(); // Reset progress to level 0
```

Check the cubit docs for reference of all methods
